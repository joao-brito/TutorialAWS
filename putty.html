Acessando a máquina através do Putty:


Chame o programa “putty”.



Cole o IP da máquina, que você copiou, na aba “Host Name (or IP address)”.



Na aba Category vá até "SSH" e clique no "+". Clique em “Auth”.



Em “Private key file fou athentication”, clique em "Browse".



Selecione a chave (arquivo em .ppk). Clique em “Abrir”.



Clique em “Open”.



Clique em “Accept”.



Faça o login na máquina.



Dê o comando “sudo su”.



Dê o comando “apt update” para ver o que precisa ser atualizado.



Dê o comando “apt upgrade” para atualizar.



Dê o comando “apt install apache2” para instalar o Apache.
